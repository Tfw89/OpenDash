# OpenDash

OpenDash is an open-source platform for replacing or modernizing OEM motorcycle instrument displays while preserving factory functionality.

The initial development target is the 2016 Kawasaki ZX-14R center LCD.

## Goals

- Preserve OEM gauges and warning indicators
- Replace the factory LCD with a modern TFT
- Maintain a factory-quality appearance
- Build a modular platform that can support additional motorcycles

## Status

Early software development and hardware reverse engineering.
